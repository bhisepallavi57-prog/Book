# 📚 Book Information Search System

## 📌 Project Overview

The Book Information Search System is a Python-based mini project that allows users to search for books and view detailed information about them.

The system stores book details using a Python dictionary and displays information such as book descriptions, topics covered, and difficulty level when a user searches for a specific book.

---

## 🚀 Features

* Search books by name
* Case-insensitive search
* Displays detailed book information
* User-friendly interface
* Exit option available
* Uses Python dictionaries for data storage

## ▶️ How to Run

1. Open the project in  Jupyter Notebook.
2. Run the Python file:

```bash
python book_search.py
```

3. Enter a book name.
4. View the book details.
5. Type `exit` to close the application.
